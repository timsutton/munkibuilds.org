Munki auto-builds

These are not official releases, these are automated builds from the master Git branch for testing. Builds are performed on OS X Server 10.6.8, Xcode 3.2.6, using the build script provided in the project repository.

There is a download-and-install script available to get and install the tools in one step. Run the following command in a terminal session (once you've audited the script yourself):

curl https://munkibuilds.org/latest.sh | sh
